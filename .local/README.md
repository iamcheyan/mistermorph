# 本地开发工作区

这个目录用于存放本地开发相关的笔记、日志和辅助文件。

## 目录结构

```
.local/
├── README.md              # 本文件
├── notes/                 # 开发笔记、分支策略、功能清单
│   ├── branch-strategy.md     # 分支管理策略与开发工作流程
│   ├── backup-branch-features.md  # 36 个提交分类清单
│   └── development-log.md     # 开发日志
├── logs/                  # 本地运行日志、调试输出
├── scripts/               # 本地辅助脚本
└── backups/               # 本地备份文件
```

## 文件说明

- **notes/**: 开发过程中的策略文档、功能清单、待办事项
- **logs/**: 程序运行日志，方便排查问题
- **scripts/**: 本地开发辅助脚本
- **backups/**: 临时备份重要文件

## 注意

这个目录已纳入版本管理，用于在多台机器间同步开发笔记。敏感信息（如 API Key）不要放入此目录。
