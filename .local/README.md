# 本地开发工作区

这个目录用于存放本地开发相关的文件，**不会被提交到 Git**。

## 目录结构

```
.local/
├── README.md          # 本文件
├── notes/             # 开发笔记、待办事项、想法记录
├── logs/              # 本地运行日志、调试输出
├── scripts/           # 本地辅助脚本
└── backups/           # 本地备份文件
```

## 使用建议

- **notes/**: 记录开发过程中的想法、问题、解决方案
- **logs/**: 保存程序运行日志，方便排查问题
- **scripts/**: 放置本地开发辅助脚本
- **backups/**: 临时备份重要文件

## 注意

这个目录已被 `.git/info/exclude` 排除，不会被提交。
